# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview

This project analyzes a pizza sales dataset using SQL to uncover key business insights such as revenue trends, customer behavior, and product performance.

The goal is to simulate a real-world business scenario and answer critical questions that can help improve sales and decision-making.

---

## 📂 Dataset Description

The dataset contains information about:

* Orders
* Order details
* Pizza types & categories
* Pricing and quantities

---

## 🎯 Business Problems Solved

### 🔹 Basic Analysis

* Retrieved the **total number of orders placed**
* Calculated the **total revenue generated**
* Identified the **highest-priced pizza**
* Found the **most common pizza size ordered**
* Listed the **top 5 most ordered pizza types**

---

### 🔹 Intermediate Analysis

* Analyzed **total quantity ordered per pizza category**
* Studied **distribution of orders by hour of the day**
* Found **category-wise pizza distribution**
* Calculated **average number of pizzas ordered per day**
* Identified **top 3 pizza types based on revenue**

---

### 🔹 Advanced Analysis

* Calculated **percentage contribution of each pizza type to total revenue**
* Analyzed **cumulative revenue over time**
* Identified **top 3 pizza types by revenue within each category**

---

## 🛠️ SQL Concepts Used

* Joins (INNER JOIN)
* Aggregations (SUM, COUNT, AVG)
* GROUP BY & ORDER BY
* Subqueries
* Window Functions (for cumulative revenue)

---

## 📊 Key Insights

* Certain pizza types contribute significantly more to revenue than others
* Peak order times occur during specific hours of the day
* A small number of pizzas drive a large portion of total sales
* Category-wise performance helps identify strong and weak segments

---

## 💡 Business Recommendations

* Focus marketing on high-revenue pizza types
* Optimize staffing during peak hours
* Promote low-performing pizzas to increase sales
* Use category insights to improve menu strategy

---

## 📌 Conclusion

This project demonstrates how SQL can be used to extract meaningful insights from raw data and support business decision-making.
